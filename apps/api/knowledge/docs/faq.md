## 可以先咨询吗 {#free-consultation}

可以。先写小纸条或问小酷，酷里会先判断需求是否适合做。咨询本身不等于正式报价。

## 什么时候需要定金 {#deposit}

开发、部署、远程排查、多轮沟通或不确定性较高的需求，可能需要先付定金。小需求在范围清楚时可能验收后付款。

## 需要准备什么材料 {#materials}

截图、源文件、报错信息、参考链接、目标效果、截止时间和预算范围都很有用。材料不完整也可以先提交。

## 哪些信息不要发 {#sensitive-info}

不要直接发送密码、验证码、私钥、支付凭证、身份证等敏感信息。确实涉及账号类操作时，由管理员另行确认安全方式。
