## 先了解酷里 {#understand-kuli}

酷里适合处理 AI 工具配置、文档处理、小工具开发、部署配置、API/token 咨询，以及暂时不知道怎么分类的临时需求。

## 写小纸条 {#write-note}

登录后可以写小纸条。你不用一次把需求写得很专业，只要先说明想完成什么、现在哪里卡住、希望最终交付成什么样。

## 查看订单 {#view-orders}

提交后可以在我的订单里查看状态、补充材料、阅读管理员公开回复、查看报价和交付物。

## 验收与继续沟通 {#acceptance}

交付后先看结果，再确认是否验收。小需求可能验收后付款，复杂需求可能需要先付定金，最终以管理员确认为准。
