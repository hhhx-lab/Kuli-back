## 联系入口 {#contact-entry}

优先通过小纸条和订单沟通区联系酷里。这样需求、附件、报价和交付记录会留在同一个订单里。

## 人工介入 {#human-support}

小酷可以先整理问题，但不会假装人工客服已经介入。需要人工确认时，它会帮你生成给管理员看的摘要。

## 响应预期 {#response-expectation}

响应时间受需求复杂度、材料完整度和管理员排期影响。紧急需求请在小纸条里明确截止时间。

## 敏感信息提醒 {#safety-reminder}

不要在文档、订单消息或小酷对话中直接发送密码、验证码、私钥和支付凭证。
