## 小纸条 {#note}

小纸条是低压力需求入口，用来让酷里先判断需求是否能做、材料是否足够、是否需要报价。

## 订单状态 {#order-status}

submitted 表示已提交，clarifying 表示需要补充材料，quoted 表示已报价，deposit_pending 表示等待定金，in_progress 表示处理中，review 表示等待验收，completed 表示已完成。

## 定金与尾款 {#deposit-final-payment}

定金用于复杂或风险较高的需求启动前确认投入。尾款通常在交付验收后确认。具体金额、节点和周期必须由管理员确认，小酷不能直接报价。

## 小酷 Agent {#xiaoku-agent}

小酷是酷里的 3D 小猫服务助手，可以解释服务、整理需求和说明订单状态，但不替代管理员承诺价格、周期或最终结果。
